# HDC Host - Landing Page de Hospedagem

Esta é uma landing page moderna e responsiva para uma empresa de serviços de hospedagem web, desenvolvida com foco em UI/UX e conversão de clientes.

## Descrição do Projeto

O projeto apresenta uma interface completa para venda de planos de hospedagem, incluindo seções de diferenciais competitivos, tabelas de preços detalhadas, verificação de disponibilidade de domínio e formulário de contato direto.

## Funcionalidades Principais

* **Navegação Intuitiva:** Navbar fixa com acesso rápido a preços, contatos e área de login.
* **Destaques de Serviço:** Seção detalhando benefícios como Segurança (Shield), Performance (Rocket) e Suporte 24/7.
* **Tabelas de Planos:** Comparativo visual entre quatro níveis de serviço (Básico, Dedicado, Dedicado Plus e Dedicado Pro).
* **Busca de Domínio:** Interface integrada para usuários verificarem a disponibilidade de nomes para seus sites.
* **Formulário de Contato:** Sistema pronto para captação de leads com campos para nome, e-mail e descrição do projeto.
* **Responsividade Completa:** Layout adaptável para dispositivos móveis, tablets e desktops através de Media Queries.

## Tecnologias Utilizadas

* **HTML5:** Estruturação semântica do conteúdo e formulários.
* **CSS3:** Estilização personalizada utilizando Flexbox para o sistema de grid e tabelas de planos.
* **Font Awesome 7.0:** Iconografia moderna para representação visual dos serviços.
* **Google Fonts (Helvetica):** Tipografia limpa para garantir legibilidade técnica.

## Estrutura de Arquivos

* `index.html`: Documento principal contendo todas as seções da landing page.
* `css/style.css`: Estilização global, variáveis de cores e configurações de responsividade.
* `img/`: Diretório contendo o logotipo e os banners de fundo da aplicação.
